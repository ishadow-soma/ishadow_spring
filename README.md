# Getting Started

## Local   
    Local Execution   
        gradlew build   
        cd ./build/libs   
        java - jar ishadow-0.0.1-SNAPSHOT.jar --spring.profiles.active=local   
        
    Local Swagger        
        http://localhost:9000/swagger-ui.html#/

## Prod    
    https://ishadow.kr    
    https://subdomain.ishadow.kr/swagger-ui.html#/
